# Демонстрации утечек памяти

Интерактивные примеры утечек памяти в JavaScript и React для обучения работе с Chrome DevTools.

## Запуск

```bash
npm install
npm run dev
```

## Демо

| № | Название | Что искать |
|---|----------|------------|
| 1 | Detached элементы | Узлы DOM, отключённые от документа, но сохранённые в памяти |
| 2 | Утечки JS + allocation on timeline | Большие строки и массивы на уровне модуля |
| 3 | Метод трёх снапшотов | Классы с растущим числом экземпляров |
| 4 | Утечки в React | Слушатели событий без отписки при unmount |
| 5 | useMemo и селекторы | Правильная мемоизация (не утечка) |

## Инструменты

- **Chrome DevTools** → вкладка Memory
- Метод трёх снапшотов
- Allocation timeline
- Performance monitor
- `queryObjects()` в консоли

## Как использовать

1. Откройте DevTools (F12)
2. Перейдите на вкладку Memory
3. Выберите тип записи (Heap snapshot / Allocation timeline)
4. Выполните действия на странице (клики, открытия модалок)
5. Сравните снапшоты или изучите timeline

## Технологии

- React 19
- TypeScript
- Vite
- CSS-переменные (тёмная тема в стиле Т-Банка)
